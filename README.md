# Exploratory Data Analysis (EDA) - Titanic Dataset

## Objective
Perform Exploratory Data Analysis (EDA) using Python to uncover patterns, trends, and relationships within the Titanic dataset.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analyses Performed
- Data inspection with .info(), .describe()
- Missing value analysis
- Value counts
- Histograms
- Boxplots
- Scatterplots
- Heatmap
- Pairplot
- Survival analysis by gender and passenger class

## Key Findings
- Female passengers had a higher survival rate.
- First-class passengers survived more often.
- Fare showed outliers.
- Most passengers were aged between 20 and 40.
- Ticket fare had a positive relationship with survival.
